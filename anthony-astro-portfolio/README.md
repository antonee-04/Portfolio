# Anthony Grummett — Astro Portfolio

Wix-like, fast Astro site wired with your socials:
- GitHub: https://github.com/antonee-04
- LinkedIn: https://www.linkedin.com/in/anthony-grummett-008917261/
- Wix: https://anthonygrummett818.wixsite.com/grummettgames

## Quick start
npm install
npm run dev

## Deploy (Vercel)
Push to GitHub, then import into Vercel. Auto-deploy on push.

## Import from Wix
- Download images/videos → /public/media/
- Paste text into project pages (src/pages/projects/*)
- Put Unity WebGL builds under /public/builds/<project>/
