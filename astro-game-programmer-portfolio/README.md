# Astro Game Programmer Portfolio

A minimal, fast Astro starter tailored for a game programmer portfolio with:
- Home (hero + selected projects)
- Projects index (cards)
- Project case-study pages (DiveCraft, AI Grid, Tools)
- About / Contact
- Dark, lightweight styling
- Ready for Recipe B deployment (GitHub + Vercel)

## Quick start

```bash
npm install
npm run dev
```
Open http://localhost:4321

## Deploy (Recipe B)
1. Create a GitHub repo and push this folder.
2. On Vercel: New Project → Import from GitHub → Deploy.
3. Add a custom domain in Vercel (optional).

## Unity WebGL
Put your built WebGL folder under `public/builds/<project>/` and link it from a project page.
Use gzip compression for widest compatibility.
